# Política de Segurança

## Versões Suportadas

Como este repositório contém principalmente arquivos de texto (prompts), não há "executáveis" tradicionais. No entanto, mantemos as versões da interface web atualizadas.

| Versão | Suportada |
| :--- | :--- |
| 1.3.x | ✅ Sim |
| < 1.3.0 | ❌ Não |

## Reportando uma Vulnerabilidade

Se você encontrar uma vulnerabilidade na interface web ou uma falha de segurança em um dos prompts (ex: um prompt que induz a IA a gerar código malicioso sem aviso), por favor não abra uma Issue pública.

Envie um e-mail para **gabriel.cardoso@datalin.com.br**.

Daremos retorno em até 48 horas úteis sobre os próximos passos.
